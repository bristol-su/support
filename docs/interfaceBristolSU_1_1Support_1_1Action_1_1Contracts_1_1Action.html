static BristolSU\Support\Action\Contracts\Action::getFieldMetaData ( )
static

Register metadata about the fields the action needs.

For each action field requested in getFields, pass in a label and a helptext. e.g. [ 'to' => [ 'label' => 'To', 'helptext' => 'Email address of the recipient' ], ... ]

BristolSU\Support\Action\Contracts\Action::getFields ( )

Get the fields that the action needs.

If the action needs any settings, such as a message, you should return them here along with a default. e.g. [ 'to' => 'examp.nosp@m.le@e.nosp@m.xampl.nosp@m.e.co.nosp@m.m', 'subject' => 'Default Subject', 'message' => 'Default message' ]
