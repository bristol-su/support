getFields() : array

Get the fields that the action needs.

If the action needs any settings, such as a message, you should return them here along with a default. e.g. [ 'to' => 'example@example.com', 'subject' => 'Default Subject', 'message' => 'Default message' ]

getFieldMetaData()

getFieldMetaData() : array

Register metadata about the fields the action needs.

For each action field requested in getFields, pass in a label and a helptext. e.g. [ 'to' => [ 'label' => 'To', 'helptext' => 'Email address of the recipient' ], ... ]
